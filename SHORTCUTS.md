# Usage

## Short Cuts

* w: 绘制矩形框
* f: 绘制点
* f -> <LeftButton>: 绘制前景点
* f -> <RightButton>: 绘制背景点
* r -> 执行分割算法
* s: 显示分割结果
* m: 显示ground truth
* Esc: 取消绘制并恢复鼠标
* <Space>: 显示精度
* 滚轮: 查看不同切片
* Ctrl + 滚轮: 缩放
* Shift + 滚轮: 上下移动视图
* Ctrl + s: 保存绘制的点和矩形框
* 双击文件名打开文件
* Del: 删除选中的点或矩形框

## 编辑矩形框的坐标

* 编辑 z1 和 z2 坐标以限制分割的范围
* 选中记录, 按 Del 删除点或矩形框
